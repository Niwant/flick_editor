# Quick Start Guide

## Installation

```bash
npm install
```

## Running the Application

```bash
npm run dev
```

The application will start at `http://localhost:3000`

## First Steps

1. **Import Media**: Click "Import Media" button in the toolbar to add video, audio, or image files
2. **Edit on Timeline**: Drag clips on the timeline to reposition them
3. **Split Clips**: Select a clip and press `S` or use the Split tool to cut at the playhead
4. **Canvas Editing**: Switch to Canvas view to position and transform elements visually
5. **Node Graph**: Switch to the Node Graph tab to create visual effect compositions

## Keyboard Shortcuts

- `Space` - Play/Pause
- `⌘+1` - Canvas view
- `⌘+2` - Playback view  
- `⌘+3` - Split view
- `⌘+←/→` - Frame-by-frame scrubbing
- `S` - Split clip at playhead
- `Delete` - Delete selected clip
